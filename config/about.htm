gvSIG EIEL 1.0 Alpha 3

Conjunto de herramientas para la EIEL

Versión: 1.0 Alpha 3

Descripción: :

Personalización de gvSIG y conjunto de nuevas herramientas destinadas a facilitar las tareas relativas a la EIEL.

Características:

  • Adaptación a imagen corporativa.
  • Conexión a la Base de Datos Territorial.
  • Gestión de usuarios y permisos de la BDT.
  • Digitalización de elementos (nuevas CADTools).
  • Visualización y edición de datos con NavTable e Formularios.
  • Carga automática de mapas configurables
